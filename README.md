# Portfolio

A comprehensive showcase of full-stack web development, machine learning, and data engineering projects.

## Projects

### 1. Online Course Management System (OCMS)
A complete learning management system built with Django and modern web technologies.

**Tech Stack:**
- Backend: Django, Django REST Framework, Python
- Frontend: Vue.js, Tailwind CSS, Vite
- Database: PostgreSQL
- Deployment: Docker

**Features:**
- User authentication and authorization (teachers, students, admin)
- Course management and content delivery
- Student enrollment and progress tracking
- Course reviews and ratings
- Responsive dashboard

**Directories:**
- `ocms/` - Django backend application
- `ocms-frontend/` - Vue.js frontend
- `static-frontend/` - Static HTML/CSS frontend

---

### 2. Resume Screening with AI
An intelligent resume screening system powered by artificial intelligence and NLP techniques.

**Tech Stack:**
- Backend: Flask, Python
- Frontend: React, Tailwind CSS, Vite
- Specialized Libraries: NLP, ML models

**Features:**
- Automated resume parsing and analysis
- Skill extraction and matching
- Candidate scoring and ranking
- Resume format handling

**Directories:**
- `backend/` - Flask API server
- `frontend/` - React-based web interface
- `resumes/` - Sample resume files

---

### 3. Smart Campus Ticket Raising Platform
A platform for managing support tickets and facility requests on campus.

**Tech Stack:**
- Framework: Django
- Language: Python

**Features:**
- Ticket creation and tracking
- Issue categorization
- Status management
- Admin dashboard

**Directories:**
- `smart_campus/` - Main Django project
- `tickets/` - Ticket app module

---

### 4. ETL Pipeline
An extract, transform, and load data pipeline for automated data processing.

**Tech Stack:**
- Language: Python

**Features:**
- Data extraction from multiple sources
- Data transformation and cleaning
- Output generation
- Logging and monitoring

**Directories:**
- `pipeline.py` - Main ETL script
- `logs/` - Pipeline execution logs
- `output/` - Cleaned data outputs

## Getting Started

### Prerequisites
- Python 3.8+
- Node.js 14+
- Docker (optional)
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/pranitarin05/portfolio
cd portfolio
```

2. For specific projects, refer to their individual README files or setup guides.
